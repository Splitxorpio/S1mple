Operators

Like Python, Operators are to add/subtract/multiply/divide two integers or floats. the '+' operator is for adding, the '-' operator is for subtracting, the '*' operator is for multiplying, and the '/' operator is for dividing.

Loops

A for loop is a python statement that loops what ever is inside of it a certain amount of times. This is extremely useful if you find yourself repeating code.

In this case, it will print brandon 7 times, since there are seven characters in 'brandon'

The range function helps python understand how many times to print something, so if you do "for i in range(name)" it will tell python to check how many characters are in the name variable, same goes for numbers, if you do "for i in range(10)", python will know to repeat whatver is in the for loop 10 times.
